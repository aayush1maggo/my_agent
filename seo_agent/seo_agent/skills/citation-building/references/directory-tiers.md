# Citation Directory Priority Tiers

## Tier 1 — Must Have (Highest ranking impact)
Submit to these first. These are the most authoritative and have the most direct local ranking influence.

| Directory | URL | Notes |
|---|---|---|
| Google Business Profile | business.google.com | Most important — verify by postcard/phone/video |
| Bing Places for Business | bingplaces.com | Import from Google or create manually |
| Apple Maps / Apple Business Connect | businessconnect.apple.com | Requires Apple ID, verify by phone |
| Yelp | biz.yelp.com.au | Claim existing listing if it exists |
| Facebook Business | facebook.com/pages/create | Set as local business category |
| Yellow Pages Australia | yellowpages.com.au | High DA Australian directory |

---

## Tier 2 — High Priority (Strong local signal)
