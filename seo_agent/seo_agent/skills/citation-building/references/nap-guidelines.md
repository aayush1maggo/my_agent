# NAP Consistency Guidelines

NAP = Name, Address, Phone. Inconsistency across citations dilutes local ranking signals.

---

## Business Name Rules

- Use the exact legal trading name — no keyword stuffing
  - BAD: "Joe's Plumbing Melbourne Emergency Plumber"
  - GOOD: "Joe's Plumbing"
- Do not include location in the business name unless it is legally part of the name
- Use the same capitalisation everywhere (typically Title Case)
- Choose one abbreviation style and stick to it: "St" not "Street" not "St." — pick one

---

## Address Format (Australian Standard)

```
[Street Number] [Street Name] [Street Type],
[Suburb], [State Abbreviation] [Postcode]
```

Example:
```
42 Collins Street,
Melbourne, VIC 3000
```

Rules:
- Use abbreviated state: VIC / NSW / QLD / SA / WA / TAS / ACT / NT
- Include postcode (4 digits for AU)
- No trailing comma after suburb/state line
- No "Australia" at end for domestic directories (add for international ones)
- PO Box addresses should NOT be used for citation building — use physical address only

---

## Phone Number Format (Australian)

Standard format: **(03) 9XXX XXXX** (Melbourne landline example)

| Type | Format | Example |
|---|---|---|
| Melbourne landline | (03) 9XXX XXXX | (03) 9876 5432 |
| Sydney landline | (02) 9XXX XXXX | (02) 9123 4567 |
| Mobile | 04XX XXX XXX | 0412 345 678 |
| 1300 number | 1300 XXX XXX | 1300 123 456 |
| 1800 number | 1800 XXX XXX | 1800 987 654 |

Rules:
- Use brackets around area code: (03) not 03
- Space after area code, space in middle of 8-digit number
- Never use dashes or dots as separators
- Use the same number across ALL citations — do not mix mobile and landline

---

## Website URL Format

- Always use HTTPS
- No trailing slash: `https://example.com.au` not `https://example.com.au/`
- Use the canonical version (with or without www — be consistent)
- For citations, add UTM tracking: `https://example.com.au?utm_source=yelp&utm_medium=local-listing`
- Do not include UTM in the canonical URL used for schema markup

---

## Description Variants

Prepare 3 versions so each directory gets unique content:

**Short (< 150 chars):** For directories with tight limits
> "Licensed Melbourne plumber offering emergency repairs, hot water systems, and drainage services. Available 7 days."

**Medium (150–300 chars):** Standard directories
> "Joe's Plumbing provides fast, reliable plumbing services across Melbourne. Specialising in emergency call-outs, hot water system repairs, blocked drains, and new installations. Fully licensed and insured. Same-day service available."

**Long (300–750 words):** Premium profiles (Google, Yelp, Bing Places)
> Full description covering: who you are, what you do, your service area, your differentiators, key services (mention 3–5 with keywords naturally), call to action.

---

## Common NAP Mistakes to Flag and Fix

| Mistake | Example | Fix |
|---|---|---|
| Name variation | "Joe's Plumbing Services" vs "Joes Plumbing" | Standardise to legal name |
| Address abbreviation inconsistency | "St" vs "Street" | Pick one, apply everywhere |
| Phone format inconsistency | "03 9876 5432" vs "(03) 9876-5432" | Use canonical format |
| Old address still live | Previous office address | Update or remove old listing |
| Keyword-stuffed name | "Best Plumber Melbourne Joe's Plumbing" | Remove keyword stuffing |
| URL mismatch | HTTP vs HTTPS, www vs non-www | Use canonical HTTPS version |
