## Python Course Curriculum: Logic Building to Advanced Scripting

The curriculum is structured to build your skills progressively. Every concept is reinforced through coding exercises and projects, ensuring you can apply what you learn — not just recall it.

### Fundamentals and Control Flow

The foundation of every Python program is clean, logical code. This module ensures you build strong habits from the start:

- **Python 3.x environment setup** — installing Python, configuring virtual environments, and choosing the right IDE (VS Code, PyCharm).
- **Variables, data types, and operators** — integers, floats, strings, booleans, and type conversion. Understanding Python's dynamic typing and how it differs from statically typed languages.
- **Control flow** — if/elif/else statements, for loops, while loops, break, continue, and nested loops. Emphasis on writing efficient loops rather than brute-force approaches.
- **Functions and scope** — defining functions, parameters, return values, default arguments, *args and **kwargs, and understanding local vs. global scope.
- **Error handling** — try/except/finally blocks, custom exceptions, and defensive programming practices.
- **File I/O** — reading from and writing to text files, CSV files, and JSON files.
- **Modules and packages** — importing standard library modules, creating your own modules, and managing dependencies with pip.

By the end of this module, you will be comfortable writing Python scripts that take input, process data, handle errors gracefully, and produce meaningful output.

### Mastering Data Structures and Algorithmic Thinking

This is the module that separates job-ready developers from those who only know syntax. Every data structure is taught with its real-world applications and its performance characteristics.

**Linear Data Structures:**
- **Arrays and Lists** — dynamic arrays in Python, list operations, time complexity of common operations (append, insert, delete, search).
- **Strings** — string manipulation techniques, pattern matching, and common interview problems (anagrams, palindromes, substring search).
- **Stacks** — implementation using lists, applications in expression evaluation, bracket matching, and undo operations.
- **Queues** — standard queues, double-ended queues (deque), and priority queues. Applications in BFS, task scheduling, and buffering.
- **Linked Lists** — singly linked lists, doubly linked lists, and circular linked lists. Pointer manipulation, reversal, cycle detection, and merge operations.

**Non-Linear Data Structures:**
- **Trees** — binary trees, binary search trees (BST), AVL trees, and heaps. Traversal algorithms (inorder, preorder, postorder, level-order). Applications in hierarchical data representation and priority scheduling.
- **Graphs** — adjacency matrix and adjacency list representations. BFS, DFS, Dijkstra's algorithm, and topological sorting. Applications in social networks, routing, and dependency resolution.
- **Hash Tables** — hashing functions, collision resolution (chaining, open addressing), and Python's dictionary implementation. Applications in caching, deduplication, and frequency counting.

**Core Algorithms:**
- **Sorting** — bubble sort, selection sort, insertion sort, merge sort, quick sort, and counting sort. Understanding when to use each algorithm based on data characteristics.
- **Searching** — linear search, binary search, and interpolation search. Binary search variations for rotated arrays, finding boundaries, and optimisation problems.
- **Recursion and backtracking** — recursive problem decomposition, memoisation, and backtracking for constraint satisfaction problems (N-Queens, Sudoku solver).
- **Dynamic programming** — identifying overlapping subproblems, building solutions bottom-up and top-down. Classic problems including longest common subsequence, knapsack, and coin change.
- **Greedy algorithms** — activity selection, Huffman coding, and understanding when greedy approaches produce optimal solutions.
- **Time and space complexity analysis** — Big O, Big Theta, and Big Omega notation. Analysing nested loops, recursive calls, and amortised complexity.

Each algorithm topic includes at least 10–15 practice problems graded by difficulty, drawn from platforms like LeetCode and HackerRank, ensuring you build both understanding and speed.

Python's strength extends far beyond algorithms. This module covers the practical applications that make Python developers valuable in any organisation:

- **Object-Oriented Programming (OOP)** — classes, objects, inheritance, polymorphism, encapsulation, and abstraction. Design patterns commonly used in Python (Singleton, Factory, Observer).
- **Web scraping with BeautifulSoup and Requests** — extracting data from websites, handling pagination, managing request headers, and respecting robots.txt. You will build a scraper that collects and structures data from real websites.
- **Automation with Python** — automating file operations, email sending, PDF generation, and Excel manipulation with openpyxl. These are the scripts that save teams hours of manual work every week.
- **Data handling with NumPy and Pandas** — array operations, DataFrame manipulation, data cleaning, filtering, grouping, and basic statistical analysis. Enough to work with real datasets and produce meaningful insights.
- **GUI development with Tkinter** — building desktop applications with graphical interfaces. Creating forms, buttons, menus, and event handlers for standalone tools.
- **Introduction to web frameworks** — a brief introduction to Flask and Django, showing how Python powers the backend of web applications. This provides a roadmap for further specialisation after the course.
